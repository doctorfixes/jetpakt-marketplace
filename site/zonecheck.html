Know the Risk Before You Build

ZoneCheck classifies any US address against FEMA flood zones and enriches it with neighborhood data — school ratings, crime stats, transit access, and more — in one API call.

Try It Now API Docs

What You Get

🏠 FEMA Flood Zone Classification

Map any address to FEMA flood risk: SFHA high-risk zones (A, AE, AO, VE), moderate-risk (B, X500), and low-risk (C, X), plus LOMR letter status.

📊 School Ratings

GreatSchools ratings for nearby elementary, middle, and high schools. Average scores and distance-weighted proximity for each tier.

🚔 Crime & Safety Stats

Per-capita crime rates at the neighborhood level — property crime, violent crime, and an overall safety score relative to national averages.

🚇 Transit & Walk Access

Walk Score-like transit accessibility: bus stops, rail stations, bike lanes within 1 mile, plus a composite mobility score.

⚡ Supabase Edge Function

Deployed as a Deno Edge Function on Supabase — cold starts under 2s, hot-path responses in under 1s. Free tier handles 50 requests/month.

🔗 DealScore Integration

ZoneCheck powers DealScore's neighborhood enrichment layer automatically. Every property scored by DealScore passes through ZoneCheck.

API Usage

Send a US address via POST. Get back flood risk, neighborhood data, and metadata in under 2 seconds.

# POST /zonecheck
curl -X POST \
  -H "Content-Type: application/json" \
  -d '{
  "address": "1600 Pennsylvania Ave NW, Washington, DC 20500"
}' \
  "https://htnufvbzsfdfadnnfnje.supabase.co/functions/v1/zonecheck"
// Example response (truncated)
{
  "flood_zone": "X",
  "flood_risk": "Low",
  "school_rating": { "average": 7.2, "elementary": 8, "secondary": 6 },
  "crime": { "property_per_capita": 0.032, "violent_per_capita": 0.006, "safety_score": 72 },
  "transit": { "stops_within_1mi": 14, "mobility_score": 68 }
}
