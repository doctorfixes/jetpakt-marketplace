# JetPakt Pulse (restaurant business — live)

JetPakt is a real-time operational command layer for restaurants.
It converts fragmented data into actionable alerts.

Core loop:

Signal → Insight → Action

Examples:
- Sales spike + low labor → alert
- Complaints spike → alert
- Ticket times rise → alert

JetPakt focuses on preventing bad shifts in real time.

# JetPakt AI (AI Agency — new, this pivot)

JetPakt AI is an AI front-desk service for local businesses that have no
automated phone, booking, or web presence today: salons, dry cleaners,
roofers, plumbers, and other trades in the 80134 area (Parker, CO) and
surrounding south-metro Denver zips.

Core loop:

Call comes in → AI answers → books the appointment → confirms by text

The AI phone agent runs 24/7, so a missed call never becomes a lost job.
Optional add-ons layer on top: a one-page website + Google Business Profile,
automatic post-job review requests, and web/SMS lead intake with instant
quotes for standard jobs.

JetPakt AI focuses on the businesses an agency-built automation stack
usually skips — one-truck plumbers, one-chair salons, family-run dry
cleaners — not enterprise software buyers.

See docs/AI_AGENCY_PLAN.md for the full plan (target list, CRM design,
sales process, pricing, and milestones). Both businesses share the same
underlying CRM/outreach tooling (jetpakt_cli/) via a profile switch — see
the root README for details.
